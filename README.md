# Noir Bloom Studio Demo

Premium beauty studio demo built for portfolio, social media showcases and client acquisition.

## Stack

- Next.js 16
- TypeScript
- Tailwind CSS 4

## Pages

- `/`
- `/services`
- `/gallery`
- `/contact`

## Local development

```bash
npm install
npm run dev
```

## Quality checks

```bash
npm run lint
npm run build
```
